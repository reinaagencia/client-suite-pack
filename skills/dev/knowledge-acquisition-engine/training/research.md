# Research: Knowledge Acquisition Engine

## Fuentes consultadas
1. Voyager (Wang et al. 2023) — Automatic curriculum, skill library, iterative prompting
2. Reflexion (Shinn et al. 2023) — Episodic memory, verbal reinforcement learning
3. Lilian Weng — LLM Agents overview (planning, memory, tool use)
4. RAG papers — Hybrid search, chunking strategies, MIPS algorithms
5. Generative Agents (Park et al. 2023) — Memory stream, reflection, planning
6. AutoGPT — Autonomous task decomposition
7. ReAct (Yao et al. 2023) — Reasoning + Acting interleaved

## Técnicas clave identificadas
- Multi-source parallel research con fallback entre cuentas
- Semantic chunking vs fixed-token chunking
- Hybrid search = vector + keyword + reranker
- HNSW vs FAISS vs ScaNN para velocidad de retrieval
- Voyager-style curriculum generation basado en mastery
- Confidence scoring multi-factor
