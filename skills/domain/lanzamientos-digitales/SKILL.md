## 1. Frameworks de Lanzamiento

### 1.1 Product Launch Formula — Jeff Walker

**Filosofía**: El lanzamiento es una conversación, no un comunicado. La Secuencia de la Historia es el corazón.

**Las 4 Fases**:

| Fase | Duración | Objetivo | Actividades Clave |
|------|----------|----------|-------------------|
| **Pre-Prelanzamiento** | 2-4 semanas | Construir anticipación | Content seeding, lead magnets, lista de espera |
| **Prelanzamiento** | 5-7 días | La Secuencia de la Historia | 3-5 emails/videos contando la historia |
| **Lanzamiento** | 3-5 días | Apertura de carrito, ventas | Escasez, bonos, urgencia, lives |
| **Post-Lanzamiento** | 3-7 días | Seguimiento, cierre | No-compradores, retargeting, upsells |

**La Secuencia de la Historia** (corazón de PLF):

```
Email 1: El Problema / La Lucha
  → Describe el dolor, el obstáculo, la frustración
  → Conecta emocionalmente con la audiencia
  → "Yo también estuve ahí"

Email 2: El Punto de Inflexión
  → Qué cambió, qué descubriste
  → El momento "ajá"
  → "Y entonces encontré esto..."

Email 3: La Solución (El Producto)
  → Presenta tu oferta como la respuesta natural
  → No es "cómpralo", es "esto fue lo que me funcionó"
  → CTA a la página de ventas
```

**Tipos de Lanzamiento PLF**:

| Tipo | Descripción | Riesgo | Escala |
|------|-------------|--------|--------|
| **Seed Launch** | A tu lista existente, sin JVs | Bajo | 1-5% de la lista |
| **JV Launch** | Con socios/afiliados promoviendo | Medio | 10x-100x seed |
| **Hybrid Launch** | Combinación de ambos | Medio-Alto | Máximo potencial |

### 1.2 The Perfect Launch® — Álvaro Luque

**Filosofía**: Un lanzamiento es un EVENTO con UNA SOLA OFERTA y UNA SOLA CONVERSACIÓN.

**Las 5 Etapas**:

```
Etapa 1: CREACIÓN (Días 1-14)
  Activos: Lead magnet, squeeze page, thank you page, grupos WhatsApp
  Métrica: Tasa de conversión visita → registro
  
Etapa 2: CAPTACIÓN (Días 15-28)
  Activos: PUV, squeeze page de captación, anuncios, secuencia de invitación
  Métrica: CPL, leads/día, tasa de conversión
  
Etapa 3: CALENTAMIENTO
  La Secuencia del Dolor: 7 mensajes que tocan los dolores del avatar
  3 tipos de personas: No saben que tienen problema / No saben que hay solución / No saben que TÚ tienes la solución
  
Etapa 4: CONSUMO
  3 Clases en vivo: Clase #1 (Framework) → Clase #2 (Viaje del Héroe) → Clase #3 (La Venta)
  
Etapa 5: COMPRA
  4 Secuencias: Apertura → BAR (Bonos+Autoridad+Razones) → Lógica → Últimas Horas
```

**La Secuencia del Dolor** (7 mensajes de calentamiento):

| # | Mensaje | Propósito |
|---|---------|-----------|
| 1 | **Bienvenida** | Pautas, anticipación, sorpresa, horarios |
| 2 | **El Enemigo** | Destino → Dolores → Enemigo común → Héroe → Resultados → CTA |
| 3 | **Momento Éxito** | Historia personal → Resultados → Invitación → PUV |
| 4 | **La Polaridad** | Contraste: Viejo vs Nuevo, Complejo vs Sencillo |
| 5 | **Mi Peor Error** | Vulnerabilidad → Conexión → Aprendizaje → Invitación |
| 6 | **El Mito** | Derribar creencias limitantes del nicho |
| 7 | **El Gran Dolor** | El dolor más profundo → la solución → CTA a Clase #1 |

**Mental Trigger Box** (15 disparadores psicológicos):

| # | Trigger | Uso en Lanzamiento |
|---|---------|-------------------|
| 1 | Escasez | Cupos limitados, tiempo limitado |
| 2 | Prueba Social | Testimonios, X compradores |
| 3 | Autoridad | Credenciales, logros, resultados |
| 4 | Anticipación | "Muy pronto", adelantos |
| 5 | Enemigo Común | El dolor, el sistema antiguo |
| 6 | Prueba | Datos, resultados concretos |
| 7 | Reciprocidad | Dar valor gratis (lead magnet) |
| 8 | Sinceridad | Autenticidad, vulnerabilidad |
| 9 | Simpatía | Ser agradable, cercano |
| 10 | Especificidad | Ser concreto, detallado |
| 11 | Compromiso | Pequeños compromisos que escalan |
| 12 | Pertenencia | Comunidad, grupo exclusivo |
| 13 | Exclusividad | Solo para este grupo |
| 14 | Historia | Storytelling personal |
| 15 | Framework | Un sistema probado |

**Las 4 Secuencias de Compra**:

| Secuencia | Contenido | Timing |
|-----------|-----------|--------|
| **Apertura** | Oferta, urgencia inicial | Día 1-2 |
| **BAR** | Bonos (más valor) + Autoridad (quién eres) + Razones (por qué ahora) | Día 2-3 |
| **Lógica** | Argumentos racionales, comparación precio/valor | Día 3-4 |
| **Últimas Horas** | Cuenta regresiva, escasez real | Último día |

### 1.3 The Workshop Method — Santi Padilla

**Filosofía**: Valida ANTES de crear. Mini launch orgánico → Workshop en vivo → Upsell.

**El Ciclo**:
```
1. Anunciar la idea en Stories → generar expectativa
2. Encuesta de interés → medir demanda
3. DM a interesados → link de pre-venta
4. ¿Hay ventas? → CREA el workshop. ¿No? → Cambia de idea
5. Workshop en vivo ($7-$47) → genera confianza
6. "El Siguiente Nivel" → upsell a programa premium ($497-$1997+)
```

**4 Elementos de un Workshop que se Vende**:
1. **Problema Específico** — Resuelve UN problema, no varios
2. **Promesa Clara** — El resultado debe ser evidente desde el nombre
3. **Urgencia / Escasez** — Workshop en vivo con fecha, cupos limitados
4. **Precio Accesible** — $7-$47 para eliminar fricción

**Estructura del Workshop (40 slides)**:
```
APERTURA (5 slides): Título → Historia → Autoridad → Resultado → Agenda
SECCIÓN 1 (9 slides): Enseñanza → Historia → Testimonial → Estrategia → Objeción → Actividad
SECCIÓN 2 (8 slides): Mismo patrón
SECCIÓN 3 (8 slides): Mismo patrón
EL SIGUIENTE NIVEL (9 slides): Crear deseo → Revelación → Oferta Premium → CTA
CIERRE (1 slide): Q&A
```

### 1.4 Funnel Mindset — Russell Brunson

**Filosofía**: El embudo es una máquina de conversión predecible. Cada paso prepara para el siguiente.

**El Embudo Clásico**:
```
Tráfico (ads, orgánico, JVs)
    ↓
Lead Magnet (gratis a cambio del email)
    ↓
Welcome Sequence (nurture + autoridad)
    ↓
Tripwire / Oferta de Entrada ($7-$47)
    ↓
Core Offer / Producto Principal ($97-$997+)
    ├── Upsell #1 ($47-$197)
    ├── Upsell #2 ($97-$297)
    └── Downsell (si rechaza upsell)
    ↓
Order Bump (pequeño añadido en checkout)
    ↓
One-Time Offer (OTO) post-compra
    ↓
Cross-sell (productos relacionados)
    ↓
Membresía / Comunidad (recurrencia)
```

**Tipos de Páginas**:
| Página | Propósito | Elementos Clave |
|--------|-----------|-----------------|
| **Squeeze Page** | Capturar email | Headline, PUV, formulario |
| **Sales Page** | Vender el producto | Problema → Solución → Precio → CTA |
| **Thank You Page** | Post-registro/compra | Siguiente paso, acceso |
| **Order Form** | Checkout | Precio, garantía, order bump |
| **Upsell Page** | Oferta adicional | Valor añadido, urgencia |
| **Downsell Page** | Oferta alternativa | Precio reducido, menos features |

**Estrategia de Embudos por Tipo de Producto**:

| Tipo de Producto | Embudo Recomendado |
|-----------------|-------------------|
| Curso/Programa | Lead Magnet → Webinar → Core Offer → Membresía |
| Servicio/Consultoría | Lead Magnet → Call → Core Offer → Retainer |
| Producto Físico | Tripwire → Core Offer → Upsells → Subscribe |
| Comunidad/Membresía | Lead Magnet → Trial → Suscripción |
| Evento/Taller | Lead Magnet → Workshop Live → Premium |

## 2. Estructura de Embudos

### 2.1 Embudo de Lead Magnet
```
Anuncio → Squeeze Page (Lead Magnet) → Thank You Page → Welcome Sequence
```

### 2.2 Embudo de Webinar (VSL)
```
Anuncio → VSL/Webinar Page → Thank You Page → Email Follow-up
```

### 2.3 Embudo de Desafío (Challenge)
```
Anuncio → Página de Registro → Desafío de X Días → Oferta Final
```

### 2.4 Embudo de Consultoría
```
Anuncio → Lead Magnet → Email Sequence → Call de Descubrimiento → Propuesta
```

### 2.5 Embudo Evergreen
```
Tráfico Continuo → Página de Ventas Automatizada → Secuencia de Emails → Ventas 24/7
```

### Pirámide de Precios (Price Anchoring)
```
$0     → Lead Magnet (construye lista)
$7-47  → Tripwire / Workshop (entrada)
$97-297 → Core Offer (producto principal)
$297-997 → Programa / Certificación (premium)
$997+  → Mentoring / 1:1 (high ticket)
```

**Regla**: Muestra primero el precio más alto (o la suma de todo el valor) para que el precio real parezca una ganga.

## 3. Copywriting para Lanzamientos

### 3.1 Estructura de la Sales Page

```
1. HEADLINE (3-5 segundos)
   ▶ Promesa única de valor (PUV) en una línea
   ▶ Ej: "Cómo [resultado] sin [dolor]"

2. SUBHEADLINE (10 segundos)
   ▶ Expande la promesa, añade especificidad
   ▶ Ej: "Método probado por X personas que [lograron Y]"

3. EL PROBLEMA (30 segundos)
   ▶ Describe el dolor, la frustración, el obstáculo
   ▶ Conecta emocionalmente
   ▶ "¿Te suena familiar?"

4. LA SOLUCIÓN (1 minuto)
   ▶ Presenta tu oferta
   ▶ No features, sino beneficios
   ▶ "Y si te dijera que..."

5. QUÉ INCLUYE (2 minutos)
   ▶ Módulos, lecciones, recursos
   ▶ Cada uno con su beneficio específico
   ▶ Formato: Título → Descripción → Resultado

6. BONOS (1 minuto)
   ▶ Valor añadido que inclina la balanza
   ▶ Cada bono debe tener su propio valor declarado

7. TESTIMONIOS (1 minuto)
   ▶ Prueba social con resultados específicos
   ▶ Formato: Antes → Después → Cita

8. PRECIO (30 segundos)
   ▶ Comparación valor vs precio
   ▶ Anclaje: mostrar valor total primero
   ▶ Desglose: "esto por solo $X"

9. GARANTÍA (15 segundos)
   ▶ Eliminar el riesgo
   ▶ 7, 15, 30 días — sin preguntas

10. CTA (10 segundos)
    ▶ Llamado a la acción claro
    ▶ Urgencia si aplica
    ▶ Botón grande, visible, contrastante
```

### 3.2 Fórmulas de Headlines

| Fórmula | Ejemplo |
|---------|---------|
| **Cómo [resultado] sin [dolor]** | "Cómo conseguir 100 clientes sin invertir en anuncios" |
| **El secreto de [experto] para [resultado]** | "El secreto de los top sellers para vender todos los días" |
| **[Número] Pasos para [resultado]** | "7 Pasos para lanzar tu curso online en 14 días" |
| **Adiós a [dolor], hola a [resultado]** | "Adiós a las ventas inconsistentes, hola a los lanzamientos predecibles" |
| **Lo que nadie te dice sobre [tema]** | "Lo que nadie te dice sobre vender conocimiento online" |
| **El método [nombre] para [resultado]** | "El Método Lanzador para lanzar cualquier producto" |
| **¿[Problema]? Aquí la solución** | "¿Sin ideas para tu próximo lanzamiento? Aquí la solución" |
| **Finalmente: [resultado] sin [dolor]** | "Finalmente: lanzamientos predecibles sin estrés" |

### 3.3 Fórmulas de Email para Secuencia del Dolor

**Estructura general de cada email**:
```
Línea de Asunto: [Gatillo de atención]
Preheader: [Complemento que invita a abrir]

Cuerpo:
1. Hook (primeras 2 líneas — enganchan)
2. Desarrollo (el contenido del mensaje)
3. Gatillo emocional (conexión, identificación)
4. CTA (lo que quieres que hagan)

PD: [Urgencia o recordatorio adicional]
```

**Email 1 — Bienvenida**:
```
Asunto: [Nombre], esto empieza ya
Cuerpo:
  → Alegría de tenerlos
  → Qué va a pasar (roadmap rápido)
  → Sorpresa: algo inesperado que viene
  → Fechas y horarios clave
  → CTA: guardar en calendario
```

**Email 2 — El Enemigo**:
```
Asunto: ¿Sabes cuál es tu mayor enemigo?
Cuerpo:
  → El dolor que todos sienten
  → Pero no es culpa tuya
  → El verdadero enemigo es [sistema/mentalidad/desconocimiento]
  → Buenas noticias: se puede vencer
  → CTA: Clase #1
```

**Email 3 — Momento Éxito**:
```
Asunto: El día que todo cambió
Cuerpo:
  → Historia personal
  → Antes vs Después
  → El momento de inflexión
  → "Y si yo pude, tú puedes"
  → CTA: no te pierdas la Clase #1
```

**Email 7 — El Gran Dolor**:
```
Asunto: ¿Vas a seguir igual?
Cuerpo:
  → El dolor más profundo
  → El costo de NO hacer nada
  → "Esta es tu oportunidad"
  → CTA directo a Clase #1
```

## 4. Secuencias de Email

### 4.1 Secuencia de Bienvenida (Post Lead Magnet)
```
Día 0: Entrega del lead magnet + qué esperar
Día 1: Quién soy y por qué deberías escucharme
Día 3: Caso de éxito / testimonio relevante
Día 5: Contenido de valor adicional
Día 7: Presentación suave de la oferta principal
Día 10: Último email antes del lanzamiento
```

### 4.2 Secuencia de Prelanzamiento (PLF)
```
Día -7: Email 1 — El Problema / La Lucha
Día -5: Email 2 — El Punto de Inflexión
Día -3: Email 3 — La Solución (presentación)
Día -1: Email de anticipación — "Mañana abre"
Día 0: Email de APERTURA — enlace a la página de ventas
```

### 4.3 Secuencia de Venta (Apertura de Carrito)
```
Día 0 (apertura): "Ya está abierto" — enlace a sales page
Día 1: Secuencia BAR — Bonos
Día 2: Secuencia BAR — Autoridad
Día 3: Secuencia Lógica — Comparación precio/valor
Día 4: Últimas Horas — Escasez + urgencia
Día 5 (cierre): "Se cierra en 12 horas"
```

### 4.4 Secuencia Post-Compra
```
Inmediato: Confirmación + acceso + próximos pasos
Día 1: Bienvenida al programa
Día 3: Cómo sacar el máximo provecho
Día 7: Check-in ¿cómo vas?
Día 14: Testimonial request
Día 30: Upsell a programa avanzado
```

### 4.5 Secuencia No-Compradores
```
Día 1: "¿Tuviste algún problema con el pago?"
Día 3: Última oportunidad con bonus especial
Día 5: Historia de alguien que sí compró
Día 7: Cierre — "La puerta aún está abierta"
Día 30: Re-engagement — nuevo contenido de valor
```

## 6. Métricas y KPIs

### 6.1 Métricas por Etapa del Embudo

| Etapa | Métrica | Fórmula | Benchmark |
|-------|---------|---------|-----------|
| **Tráfico** | CPC | Costo / Clics | < $1.00 |
| | CTR | Clics / Impresiones | 1-5% |
| **Lead Magnet** | Conversion Rate | Registros / Visitantes | 20-50% |
| | CPL | Inversión / Leads | < $3 |
| **Calentamiento** | Open Rate | Opens / Entregados | 20-40% |
| | CTR Email | Clics / Opens | 3-15% |
| **Venta** | Conversion Rate | Ventas / Visitantes | 1-5% |
| | CPA | Inversión / Ventas | < 30% del precio |
| | AOV | Revenue / Ventas | Depende del producto |
| **Post-Venta** | Refund Rate | Reembolsos / Ventas | 3-10% |
| | Upsell Rate | Upsells / Ventas | 20-40% |

### 6.2 Fórmulas Clave

```
Revenue = Tráfico × Conversion Rate × AOV

ROAS = Revenue / Inversión en Ads
     → Benchmark: > 3x

CPA = Inversión Total / Clientes
    → Debe ser < 30% del precio del producto

LTV = AOV × Compras Recurrentes × Vida Media del Cliente
    → Debe ser > 3× CPA

Margen Neto = Revenue - (COGS + Marketing + Comisiones + Gastos Fijos)
```

## 8. Checklists

### 8.1 Checklist Pre-Lanzamiento

```
□ Definir producto y precio final
□ Crear lead magnet (si aplica)
□ Configurar squeeze page + thank you page
□ Escribir sales page completa
□ Configurar checkout (Hotmart, Kiwify, etc.)
□ Crear secuencia de emails completa
□ Instalar píxeles de tracking (Meta, Google, TikTok)
□ Crear anuncios (mínimo 3 variaciones por campaña)
□ Preparar materiales para afiliados (si aplica)
□ Configurar upsells / downsells / order bumps
□ Probar todo el flujo de compra de principio a fin
□ Hacer test de email deliverability
□ Preparar grupo de WhatsApp (si aplica)
□ Calendarizar lives / webinars / clases
□ Preparar dashboard de métricas en vivo
□ Hacer pre-lanzamiento a lista pequeña para validar
□ Ajustar según feedback del test
```

### 8.2 Checklist Durante el Lanzamiento

```
CADA DÍA:
□ Revisar métricas del dashboard
□ Responder comentarios y DMs
□ Mandar email del día según secuencia
□ Revisar que los píxeles están trackeando bien
□ Monitorear salud de anuncios (CTR, CPC, CPL)

CADA 2-3 DÍAS:
□ Evaluar si escalar campañas ganadoras
□ Pausar campañas perdedoras
□ Crear variaciones de anuncios basadas en datos
□ Enviar reporte a afiliados/socios

AL CIERRE:
□ Última ronda de emails con urgencia real
□ Live de cierre (si aplica)
□ Revisar métricas finales
```

### 8.3 Checklist Post-Lanzamiento

```
INMEDIATO:
□ Secuencia post-compra para compradores
□ Secuencia para no-compradores
□ Agradecimiento a afiliados/socios
□ Reporte final a afiliados

SEMANA 1:
□ Onboarding de nuevos clientes
□ Encuesta de satisfacción
□ Primer check-in con clientes

SEMANA 2-4:
□ Post-mortem del lanzamiento (qué funcionó, qué no)
□ Calcular métricas finales (Revenue, CPA, ROAS, CR)
□ Documentar aprendizajes para el próximo lanzamiento
□ Plan de retención y upsells para clientes
□ Decidir si hacer re-run del lanzamiento
```

## 9. Glosario

| Término | Definición |
|---------|-----------|
| **Lead Magnet** | Recurso gratuito de alto valor para capturar emails |
| **PUV** | Promesa Única de Valor — qué hace única a tu oferta |
| **CTA** | Call to Action — llamado a la acción |
| **Squeeze Page** | Página diseñada para una sola acción: capturar email |
| **Sales Page** | Página de ventas con toda la argumentación |
| **Thank You Page** | Página post-registro o post-compra |
| **Order Bump** | Oferta adicional en el checkout |
| **OTO** | One-Time Offer — oferta única post-compra |
| **JV** | Joint Venture — socio/afiliado que promueve tu oferta |
| **CPL** | Costo por Lead — inversión para conseguir un lead |
| **CPA** | Costo por Adquisición — inversión para conseguir un cliente |
| **CAC** | Customer Acquisition Cost — sinónimo de CPA |
| **CR** | Conversion Rate — tasa de conversión |
| **AOV** | Average Order Value — valor promedio del pedido |
| **LTV** | Lifetime Value — valor del cliente a largo plazo |
| **ROAS** | Return on Ad Spend — retorno de inversión publicitaria |
| **ROI** | Return on Investment — retorno de inversión total |
| **BAR** | Bonos + Autoridad + Razones — secuencia de venta TPL |
| **VSL** | Video Sales Letter — video de ventas |
| **TOF** | Top of Funnel — parte alta del embudo |
| **MOF** | Middle of Funnel — parte media del embudo |
| **BOF** | Bottom of Funnel — parte baja del embudo |
| **Evergreen** | Lanzamiento automatizado que corre 24/7 |
| **Scarcity** | Escasez — recurso limitado |
| **Price Anchoring** | Anclaje de precio — mostrar valor alto primero |
| **FOMO** | Fear Of Missing Out — miedo a perderse algo |
| **Retargeting** | Volver a mostrar anuncios a quienes ya interactuaron |
| **Lookalike** | Audiencia similar a tus mejores clientes |
| **UGC** | User Generated Content — contenido creado por usuarios |
| **NPS** | Net Promoter Score — lealtad del cliente |
